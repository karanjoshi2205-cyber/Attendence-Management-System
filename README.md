# Attendence-Management-System
Attendance Management System (Using C language)
This is a C program made to manage attendance of students.
This program allows the teacher to add attendance, view records and search records using ID or date.
All the data is stored in a file named "attendance.txt", so the records are saved even after closing the program.
When the program runs, it shows a menu on the screen. The user can choose different options like marking attendance, viewing records or searching data. Based on the input, the program performs the required task.
This project uses basic concepts of C such as structure, file handling, functions and loops.
There are some limitations in this program also. It does not include any security system.
Overall, this is a basic project to understand how file handling and data management works in C language.
